Installation
---

On EE1.x the necessary hooks for one-click installation don't exist.

To install you'll need to add the following code to a CORE file.

---

$theme_url = $PREFS->ini('theme_folder_url'). 'third_party/graphite/js/';
$buffer .= "<div class='graphite_template_logging'>";

$buffer .= '<script type="text/javascript" src="'. $theme_url . 'jquery.js"></script>';
$buffer .= '<script type="text/javascript" src="https://www.google.com/jsapi"></script>'; 
$buffer .= '<script type="text/javascript" src="'. $theme_url . 'graphite.js"></script>';
$buffer .= "</div>";

$buffer .= '<div style="background-color : #ededed;">';
$buffer .= '<div style="text-align: left; font-family: Sans-serif; font-size: 11px; margin: 12px; padding: 6px"><hr size="1"><b>TEMPLATE EXECUTION GRAPH</b><hr size="1"></div>';
$buffer .= '<div id="graphite_log_graph_holder" style="background:#000; margin-bottom : 50px;"></div></div>';
